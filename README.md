# UAV Integrated Water Sampler & Sensor Monitoring Unit

Developed for **Smart India Hackathon 2025 (SIH '25)** under the **Ministry of Jal Shakti**.

An autonomous UAV-based river monitoring system capable of:
- Geo-tagged water sampling
- Real-time water quality sensing
- Autonomous mission planning
- Remote environmental surveillance

---

# 🌊 Project Overview

This project aims to modernize river and water body monitoring using a custom-built **Hexacopter UAV** integrated with:
- Automated water sampling mechanism
- Water quality sensing unit
- GPS-based location tracking
- Autonomous flight operations

The system enables safe, fast, and contamination-free collection of water samples from difficult-to-access locations.

---

# ✨ Features

## 🚁 Hexacopter UAV
- Custom-built high-endurance hexacopter
- Stable flight with payload support
- Autonomous waypoint navigation
- Optimized frame for sensor integration

## 💧 Water Sampling Module
- Automated sample collection system
- Contamination-free operation
- Corrosion-resistant design
- Multi-location water sampling capability

## 📡 Sensor Monitoring
- Real-time water quality sensing
- Geo-tagged data collection
- Environmental monitoring support
- Live telemetry transmission

# 🛠️ Hardware Used

| Component | Description |
|---|---|

| GPS Module | Autonomous navigation |
| Hexacopter Frame | High payload UAV |
| BLDC Motors + ESCs | Flight propulsion |
| Water Pump / Valve System | Sample collection |
| Water Quality Sensors | pH / TDS / Temperature |
| Telemetry Module | Data communication |
| Battery Pack | Long endurance flight |

---

# ⚙️ System Workflow

1. UAV follows predefined GPS waypoints
2. UAV hovers above target water body
3. Sampling module collects water automatically
4. Sensors capture water quality parameters
5. GPS coordinates are tagged with collected data
6. Data transmitted to ground station
